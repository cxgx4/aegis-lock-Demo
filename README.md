# Aegis-Lock Application

This project is a React application set up with Vite, Tailwind CSS, and shadcn-ui.

## Getting Started

Follow these steps to run the application locally:

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
